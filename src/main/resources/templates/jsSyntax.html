JS Syntax

var x, y; // Cách khai báo biến

x = 5; y = 6; // Cách gán giá trị

z = x + y; // Cách tính giá trị

Cú pháp JavaScript có hai loại giá trị: Fixed values và variable values.

* Fixed values: Giá trị cố định còn được gọi là Literal.

* Variable values: Giá trị có thể thay đổi.

Mã định danh JavaScript

Trong JavaScript, mã định danh được sử dụng để đặt tên biến (và từ khóa, chức năng và nhãn). Các quy tắc cho tên hợp pháp rất giống nhau trong hầu hết các ngôn ngữ lập trình. Trong JavaScript, ký tự đầu tiên phải là một chữ cái hoặc dấu gạch dưới (_) hoặc ký hiệu đô la ($). Các ký tự tiếp theo có thể là chữ cái, chữ số, dấu gạch dưới hoặc ký hiệu đô la. Tất cả các định danh JavaScript đều phân biệt chữ hoa chữ thường .
